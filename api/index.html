The API is a public JSON API that contains all of the information on are normal website. No key is required

How to Use

Basically, you can get to the API version of our website by putting "api." before the domain name of any url, and replacing the last slash with ".json". For example, the API version of the url "https://basisahwatukee.theater/BenjiStokman/" would be "https://api.basisahwatukee.theater/BenjiStokman.json". The API is self explanitory, and there really isn't any use for it outside of of our stuff, so it won't be explained.
